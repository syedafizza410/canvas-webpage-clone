# Canvas Webpage Clone

This project is a static HTML, CSS, and JavaScript clone of the Canva homepage. It was created as a front-end development exercise to replicate the visual design and layout of the Canva website.

## Features

*   Responsive navigation bar (implemented with JavaScript for mobile behavior)
*   Hero section with a call-to-action
*   Static implementation of the Canva homepage design

## Technologies Used

*   HTML
*   CSS
*   JavaScript
